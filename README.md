# Peto's Greek Cuisine Website

Static local-business website for Peto's Greek Cuisine in Carmel, Indiana.

## Files

- `index.html` contains the current production-ready single-page website.
- `scss/` contains the SCSS source files copied from the original design.

## Notes

- The page uses a Greek blue, white, light gray, and restrained gold brand system.
- Public business details are represented in the page content and Restaurant JSON-LD.
- The menu content and prices should be rechecked against the official menu before launch.
- Reviews link out to public review sources rather than claiming unsourced aggregate ratings.
